인스턴스 변수의 초기화: 인스턴스가 생성될 때마다 각 인스턴스 별로 초기화가 이루어진다.

>클래스 변수 초기화 순서

기본값 -> 명시적 초기화 -> 클래스 초기화 블럭

> 인스턴스 변수의 초기화 순서

기본값 -> 명시적 초기화 -> 인스턴스 초기화 블럭 -> 생성자
